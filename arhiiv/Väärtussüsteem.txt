Väärtussüsteem - Nimisõna/Noun
*väärtussüsteem, väärtussüsteemi, väärtussüsteemi; väärtussüsteemid, väärtussüsteemide, väärtussüsteeme/väärtussüsteemisid*

**Tähendused/Meanings:**
🇪🇪 **1.** ühiskonnas tunnustatud väärtushinnangute kogum ja nende toimimine
🇬🇧 **1.** value system, a set of values recognized in society and their functioning

**Näited/Examples:**
● Vana režiim ja väärtussüsteem olid end ammendanud. ||The old regime and value system had exhausted itself.||
● Kindel väärtussüsteem peaks tervet kooli läbima. ||A specific value system should permeate the whole school.||
● Mitu põlvkonda on vaja uue väärtussüsteemi tekkimiseks? ||How many generations will it take for a new value system to emerge?||
● Tänapäeval ei ole ühtset väärtussüsteemi ning moraal tundub kõrvaline. ||Today there is no single value system and morality seems peripheral.||
● Ameerika väärtussüsteemis ei ole vaene lihtsalt vaene, vaid luuser. ||In the American value system, a poor person is not just poor, but a loser.||
